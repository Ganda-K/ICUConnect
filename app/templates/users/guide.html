User Guide & Ethical Considerations

Data Sources & Accuracy

Facility Information

All hospital facility locations and basic information are sourced from the Kenya Master Health Facility Registry (KMHFR), which is the official registry of health facilities in Kenya.

ICU Bed Capacities

The ICU bed capacities shown in this system are simulated for research purposes and do not reflect actual bed availability in the facilities. These numbers are used solely for demonstrating the system's functionality and should not be used for real-world decision making.

Ethical Considerations

Patient Privacy

This system is designed with patient privacy as a top priority:

  • Patient names are never displayed in full - only initials are shown
  • All patient data is stored securely and encrypted
  • Access to patient information is strictly controlled and logged
  • No patient identifiers are shared between facilities
Data Usage

Our commitment to ethical data handling:

  • Data is collected and used only for the purpose of managing ICU admissions and discharges
  • Aggregate statistics may be used for research with proper anonymization
  • No patient data is shared with third parties without explicit consent
  • All data access is logged and audited regularly

System Usage Guide

Admissions
  • Use the "New Admission" button to admit patients
  • Verify bed availability before admission
  • Enter all required patient information
  • Set appropriate priority level
Discharges
  • Select patient from the current admissions list
  • Verify patient identity before discharge
  • Complete all discharge documentation
  • Update bed status automatically
Referrals
  • Use the map interface to find available beds
  • Contact receiving facilities directly
  • Document all referral communications
  • Update patient status after transfer
Dashboard
  • Monitor bed availability in real-time
  • View admission/discharge statistics
  • Track facility capacity
  • Access historical data
Need Help?

If you need assistance or have questions about using the system, please contact your system administrator or use the "Contact Support" button available throughout the application.
